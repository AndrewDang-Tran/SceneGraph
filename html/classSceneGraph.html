Detailed Description

Purpose: Class which brings all the nodes together, and creates the scene.

Author
Andrew Dang-Tran

Constructor & Destructor Documentation

SceneGraph::SceneGraph ( )

Construct a typical SceneGraph with a root "World" node, (ID = 0) Camera Node (ID = 1) and Light Node (ID = 2) The nodes your adding should be the children of the root.

SceneGraph::~SceneGraph ( )

Makes sure to properly free the memory of all the Nodes in the Scene Graph
